비용 최적화 팁
1. 프롬프트 캐싱 (자동)
Claude Code는 프롬프트 캐싱(반복 콘텐츠 비용 절감)과 auto-compaction(컨텍스트 한계 도달 시 대화 이력 요약)을 통해 자동으로 비용을 최적화합니다. Claude
캐시된 읽기는 일반 입력 가격의 10%만 청구됩니다. 캐시 적중률 90%일 때, $100 세션이 약 $19로 줄어듭니다. Claude Code Camp
캐싱은 자동으로 켜져 있으므로 별도 설정이 필요 없습니다. 다만 5분 이상 입력이 없으면 캐시가 만료되므로, 작업 중에는 연속적으로 사용하세요.
2. CLAUDE.md 크기 제한
100줄짜리 CLAUDE.md 파일은 약 4,000 토큰을 소비하며, Claude Opus 4.6의 컨텍스트 윈도우(200,000 토큰)의 2%입니다. 전체 메모리 파일의 총합은 10,000 토큰 이하를 목표로 하세요. SFEIR Institute
제가 만든 CLAUDE.md는 약 90줄이므로 적정 범위입니다.
