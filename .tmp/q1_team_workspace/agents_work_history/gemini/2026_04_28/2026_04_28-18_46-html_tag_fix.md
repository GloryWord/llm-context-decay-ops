# HTML 태그 기호(<, >) 가독성 수정

- **작업 일시**: 2026-04-28 18:46
- **작업 목적**: HTML 코드에서 태그 시작 및 종료 기호(`<`, `>`)가 흰색(또는 밝은 색)으로 나와 보이지 않는 문제를 수정.
- **작업 내용**:
  - `editor.tokenColorCustomizations` 내 `[Visual Studio Light]` 및 `[Evergruv (Everforest) Light]` 테마의 `textMateRules`에 `punctuation.definition.tag` 관련 스코프를 추가.
  - 해당 스코프에 짙은 녹색(`#3b533e`)을 적용하여 크림색 배경 위에서 명확하게 보이도록 조치.
- **결과**: HTML 작업 시 태그와 기호들이 뚜렷하게 식별 가능해짐.
