# Q1 Final Figure Set Recommendation

## Recommended set

1. **`figures/q1_compliance_by_rule_count.png`**
   - **Purpose:** Primary Q1 figure.
   - Shows exact-cell final-turn compliance by rule count across turn counts, split into benign vs adversarial panels.
   - Best for the main slide / report section because it directly answers the Q1 question.

2. **`figures/q3_benign_vs_adversarial.png`**
   - **Purpose:** Supporting comparison figure.
   - Re-groups the same exact-cell final-turn means to make the benign vs adversarial contrast easier to read within each rule-count condition.
   - Use as a follow-up figure when the audience needs the attack-intensity split emphasized.

3. **`figures/heatmap_representative.png`**
   - **Purpose:** Sanity-check / appendix figure.
   - Gives a compact condition-level view that helps spot outliers, sparse cells, or pattern inconsistencies.
   - Best kept as a backup or appendix figure, not the lead visual.

## Not included in the core Q1 set

- **`figures/q2_per_rule_type.png`** is not part of the core Q1 bundle because it answers Q2 (rule-type analysis), not Q1.

## Short recommendation

If the final deck/report should stay tight, lead with **`q1_compliance_by_rule_count.png`** and keep **`q3_benign_vs_adversarial.png`** plus **`heatmap_representative.png`** as supporting/backup figures.
