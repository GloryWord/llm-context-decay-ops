# 작업 기록 (2026-04-28)

## 작업 개요
- 사용자 질의: `temperature=0` 설정에도 불구하고 Multi-turn 실험에서 결과가 달라지는 현상에 대한 GPT의 "Branch" 설명 검증.
- 핵심 질문: "Branch"가 무엇이며, GPT의 설명이 타당한가?

## 분석 결과 및 증거
- **데이터 검증:** `data/outputs/full_rerun_perfect_success/fast_results_hugging-quants_Meta-Llama-3.1-8B-Instruct-AWQ-INT4.jsonl` 파일 내 `exp_0017` 케이스 분석.
- **현상 확인:**
    - 반복 시행(rep) 0, 2, 3번은 8턴에서 동일한 답변을 내놓았으나, 반복 4번은 8턴에서 전혀 다른 답변(색상 리스트 나열)을 출력함.
    - 이 8턴의 출력 차이가 다음 턴의 입력 컨텍스트(History)에 포함되면서, 최종 10턴의 답변 형태(구체적 수치 vs 플레이스홀더)까지 완전히 달라짐을 확인.
- **기술적 결론:** 
    - **Branch(분기):** 대화 궤적(Trajectory)이 특정 시점의 미세한 출력 차이로 인해 서로 다른 경로로 진입함을 의미.
    - **비결정성:** GPU 연산의 부동 소수점 오차 등으로 인해 `temp=0`에서도 아주 드물게 토큰 선택이 달라질 수 있음.
    - **컨텍스트 누적:** Multi-turn 실험 특성상 앞선 턴의 오차가 뒤쪽 턴으로 전파 및 증폭됨.

## 결론
- GPT의 설명은 실제 데이터와 일치하며 기술적으로 매우 타당함.
- "Branch"는 대화의 흐름이 갈라지는 지점을 뜻하는 적절한 비유임.
