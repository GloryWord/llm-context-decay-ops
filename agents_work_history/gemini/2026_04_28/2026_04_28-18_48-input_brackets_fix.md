# 입력 중인 괄호(<, >) 및 고스트 텍스트 색상 수정

- **작업 일시**: 2026-04-28 18:48
- **작업 목적**: 사용자가 HTML 등에서 입력 중인 `<, >` 기호나 자동완성 미리보기(Ghost Text)가 흰색으로 표출되어 보이지 않는 문제를 해결.
- **작업 내용**:
  - `editorGhostText.foreground`를 차분한 올리브 그레이(`#859a88`)로 설정하여 AI 추천 코드가 잘 보이도록 함.
  - `editorBracketHighlight.foreground1~6` 색상들을 테마에 맞춘 녹색 계열들로 강제 지정하여 괄호 입력 시 흰색으로 빠지는 현상을 방지.
- **결과**: 타이핑 도중에도 모든 괄호 및 추천 텍스트가 명확하게 보임.
