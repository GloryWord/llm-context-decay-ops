# 작업 기록: final_report_case_gallery.html 계산 오류 수정 완료

## 수정된 파일
- `docs/outputs/final_report_case_gallery.html`

## 작업 내용
사용자의 요청에 따라, 실험 설계 매트릭스 표에 존재했던 수학적/논리적 계산 오류를 모두 일관성 있게 수정했습니다. 

1. **상세 시나리오 수치 통일 (4 -> 3)**:
   - T=1, T=5 조건에 잘못 기입되어 있던 시나리오 값 `4`를 원래 문서 논리에 맞게 `3`으로 전부 수정했습니다.
   - 예) `3×4×5` -> `3×3×5`
   - 예) `2×4×5` -> `2×3×5`

2. **개별 결괏값 도출(n) 재계산**:
   - R=1, 3, 5 행의 경우 모든 열의 실행 수가 `60`에서 `45`로 수정되었습니다.
   - R=7 행의 경우 모든 열의 실행 수가 `40`에서 `30`으로 수정되었습니다.

3. **Total 합계 재계산 및 변경**:
   - **Row Total(가로 합계)**: 각 R=1, 3, 5의 경우 420 -> 360, R=7의 경우 280 -> 240으로 수정했습니다.
   - **Column Total(세로 합계)**: T=1 B/A, T=5 B/A 조건의 합산이 220 -> 165 로 수정되었습니다.
   - **Grand Total(총 합계)**: 우측 최하단 총 실행 횟수 합산을 `1,540`에서 정확하게 `1,320`으로 정정했습니다.

## 결과 및 기대효과
전체 본문 설명("조합당 3개 대화 변형 × rep 5회")의 맥락과 표의 값이 100% 일치하게 되어, 문서를 읽는 사람(교수님 등)에게 혼동을 주지 않도록 완벽히 보완되었습니다.
