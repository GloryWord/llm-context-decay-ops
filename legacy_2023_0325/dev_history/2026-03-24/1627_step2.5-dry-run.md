### 3. Reasoning 모델 문제 발견 및 해결

**문제:** Qwen3.5-9B는 reasoning(thinking) 모델로, 기본적으로 긴 사고 과정을 생성
- 간단한 "Say hello" 요청에도 312 reasoning tokens 소비
- 120초 타임아웃 초과 → 전체 실패

**해결:** OpenRouter의 `reasoning: {"effort": "none"}` 파라미터로 비활성화
- Reasoning tokens: 312 → 0
- 응답 시간: 60초+ → 1~4초
